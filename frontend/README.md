# Frontend

This folder is the React + Vite frontend for the credit card fraud detection project.

## Scripts

- `npm run dev` starts the local Vite development server.
- `npm run build` creates a production build in `dist/`.
- `npm run lint` runs the frontend ESLint checks.
- `npm run preview` serves the production build locally.

## Notes

- The app submits generated CSV data to `http://127.0.0.1:8000/fraudprob`.
- Source files live in `src/` and static public assets live in `public/`.
